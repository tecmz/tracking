# 魔众一物一码溯源防伪系统

魔众一物一码溯源防伪系统支持批量生成、管理防伪码、溯源码，帮助您更好的管理商品防伪码。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
